## 使用说明

1. **judge Skill 读取**：在命名实体提取后，用此表做对照查询，命中则替换为规范中文译名。
2. **未命中处理**：保留意大利原文 + 括注初译 + 标记 `[待人审]`，详见 `docs/it-zh-bilingual-sop.md §4`。
3. **人工审阅更新**：审阅者确认新译名后，向对应表格追加条目，标注日期。
4. **计算命中率**：`glossary_hit_rate = 命中实体数 / 总实体数 × 100`，写入 `metadata.translation.glossary_hit_rate`。
