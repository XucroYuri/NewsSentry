隐私政策

最后更新：2026 年 6 月 7 日

News Sentry（以下简称"本平台"）尊重并保护用户的个人隐私。本隐私政策说明我们在您使用本平台时如何收集、使用和保护信息。

1. 信息收集

本平台为新闻聚合与情报分析工具。我们：

  • 不主动收集访问者的个人身份信息
  • 公开页面无需注册或登录即可浏览
  • 使用浏览器本地存储（localStorage）保存用户偏好设置（如语言、主题），不涉及个人信息

2. 数据来源

本平台展示的新闻内容来源于公开的 RSS 订阅源和公开 API，均注明原始出处和链接。我们不生产新闻内容，仅进行聚合、翻译和结构化处理。

3. AI 处理

本平台使用 AI 技术辅助新闻分类、翻译和重要性评估。AI 处理仅作用于公开新闻内容，不涉及用户个人信息。

4. Cookie 与本地存储

本平台使用 Service Worker 缓存静态资源以支持离线访问，使用 localStorage 保存界面偏好。不使用追踪型 Cookie。

5. 第三方服务

本平台使用 Cloudflare 提供内容分发和安全防护。Cloudflare 可能收集匿名的访问数据用于安全分析，详见 Cloudflare 隐私政策。

6. 数据安全

管理后台采用 Token 认证机制，所有 API 通信通过 HTTPS 加密。数据存储在受控服务器上，定期备份。

7. 联系方式

如有隐私相关问题，请通过 GitHub 仓库 Issues 联系我们。
